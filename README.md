# 🎫 BPO Support Ticket System
A robust Full-Stack Ticket Management tool built to streamline technical support in BPO operations.

### 🚀 Tech Stack
- **Backend:** Java, Spring Boot, Spring Data JPA
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)

### 💡 Key Features
- **Live Dashboard:** Real-time stats for Total, Open, Pending, and Solved tickets.
- **Dynamic UI:** Color-coded rows based on ticket status for better visibility.
- **Search:** Instant keyword-based search functionality.
- **Persistence:** Reliable data storage with MySQL.

### 🛠️ How to Run
1. Configure MySQL in `application.properties`.
2. Run the Spring Boot application.
3. Open `index.html` in your browser.
